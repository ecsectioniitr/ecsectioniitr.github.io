Autonomuous Quadcopter

Last year we were able to develop a remote controlled quadcopter using KK flight controller. The performance was satisfactory but the KK FC did not provide much options for us to make it autonomous. This time we are trying to use the mobile sensor data along with an arduino to stabilize the quadcopter and make its operation autonomous. Among the many uses that comes, we are majorly focussed upon its use in Indian agricultural scenario. It can be used to survey the fields and identify the diseased or damaged areas using image processing techniques.
